INTEGRASI API

"Pasang fitur login nomor WA di web kamu dengan 1 baris kode API."

1. Endpoint Ambil OTP

Gunakan metode GET untuk menarik OTP terakhir dari server GoR 1.1.

GET https://romash.vercel.app/api/get-latest?apikey=YOUR_API_KEY

2. Contoh Penggunaan (JavaScript)

async function checkOTP() {
  const res = await fetch('https://romash.vercel.app/api/get-latest?apikey=YOUR_KEY');
  const data = await res.json();
  if(data.status) {
    console.log("Kode OTP anda adalah: " + data.otp);
  }
}

3. Cara Kerja

  • User di web kamu klik "Kirim OTP".
  • Web kamu menyuruh user kirim chat ke nomor WA Provider.
  • Termux Provider menangkap angka dan mengirim ke Vercel.
  • Web kamu melakukan polling/check ke endpoint kami setiap 3 detik sampai data muncul.
